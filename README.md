# AI Resume Skill Analyzer

## Overview

AI Resume Skill Analyzer is a simple web application built with Python and Flask that analyzes PDF resumes and automatically detects technical and soft skills.

This project was created to practice Python programming, Flask web development, PDF processing, and Git/GitHub version control.

---

## Features

* Upload a PDF resume
* Extract text automatically
* Detect technical and soft skills
* Display detected skills
* Simple and user-friendly interface

---

## Technologies

* Python
* Flask
* HTML
* CSS
* pdfplumber
* Git & GitHub

---

## Installation

```bash
git clone https://github.com/tolo-shakouri/smart-cv-analyzer.git
cd smart-cv-analyzer
pip install -r requirements.txt
python app.py
```
